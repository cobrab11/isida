[help]
Показывает текущие разделы справочной системы.
[whoami]
Ваша идентификация перед ботом.
[тест]
Проверки на "живость" бота.
[test]
Проверки на "живость" бота.
[stats]
Локальная статистика посещений конференции
[gstats]
Глобальная статистика посещений конференций
[say]
Команда "Сказать". Бот выдаст в текущую конференцию всё, что будет после команды say.
[where]
Показ конференций, в которых находится бот. Так же показывается ник бота в конференции и количество участников.
[res]
Без параметра показывает топ10 рессурсов по всем конференциям, где находится бот.
С параметром - поиск по базе рессурсов.
Числа - количество рессурсов.
[serv]
Без параметра показывает все сервера, с которых заходили в конференции, где находится бот.
С параметром - поиск по базе серверов.
Числа - количество серверов.
[inbase]
Идентификация Вас в глобальной базе.
[look]
Поиск участника по конференциям, где находится бот.
[tempo]
Локальный поиск во временной базе.
[rss]
Каналы новостей:
rss show - показать текущие подписки.
rss add url time mode - добавить подписку.
rss del url - удалить подписку.
rss get url feeds mode - получить текущие новости.
rss new url feeds mode - получить только не прочтенные новости.
rss clr - удалить все новости.

url - адрес rss канала. можно задавать без http://
time - время обновления канала. число + указатель времени. H - часы, M - минуты. допускается только один указатель.
feeds - количество сообщений для получения. не более 10 шт.
mode - режим получения сообщений. full - сообщения полностью, head - только заголовки, body - только тело сообщения.
[commands]
Показывает список доступных комманд.
[uptime]
Показывает время работы бота.
[info]
Различная информация о боте.
[clear]
Скрытая очистка истории сообщений.
[talkers]
Показ локальной базы "Болтунов"
[gtalkers]
Показ глобальной базы "Болтунов"
[quit]
Завершение работы бота. Можно указать параметр, который будет показан в статусе бота при выходе.
[restart]
Перезапуск бота. Можно указать параметр, который будет показан в статусе бота при перезапуске.
[gsay]
Глобальное объявление во всех конференциях, где находится бот.
[join]
Вход в конференцию:
join room@conference.server.ru/nick - вход в конференцию room с ником nick.
join room@conference.server.ru - вход в конференцию room с последним заданным ником.
join room - вход в конференцию room на последнем заданном сервере с последним заданным ником.
[leave]
Выход из конференции:
leave - выход из текущей конференции.
leave room - выход из конференции room на последнем заданном сервере.
leave room@conference.server.ru - выход из конференции room
[rejoin]
Перезаход в конференцию.
rejoin - перезайти в текущую конференцию.
rejoin room - перезайти в конференцию room на последнем заданном сервере.
rejoin room@conference.server.ru - перезайти в конференцию room
[pass]
Временный пароль для входа в конференции.
[owner]
Работа со списком владельцев бота:
owner show - показать список владельцев.
owner add jid - добавить jid в список.
owner del jid - удалить jid из списка.
[ignore]
Работа с "Чёрным списком":
ignore show - показать список.
ignore add jid - добавить jid в список.
ignore del jid - удалить jid из списка.
[search]
Поиск по внутренней базе jid'ов.
[gtempo]
Глобальный поиск во временной базе.
[limit]
Размер сообщения, свыше которого сообщения будут разбиавться на части кратные этому размеру.
[plugin]
Управление системой плагинов:
plugin show - показать подключенные плагины.
plugin local - показать доступные для подключения плагины.
plugin add name.py - добавить плагин.
plugin del name.py - удалить плагин.
[error]
Показывает последнюю логированную ошибку или последние, если задан параметр.
[prefix]
Указание префикса комманд. Без параметра показывает текущий префикс.
[доступ]
У бота 3 уровня доступа:
0 - команды доступны всем без ограничений.
1 - доступ не ниже администратора конференции.
2 - команды управления и настроек бота. доступны только владельцу бота.
